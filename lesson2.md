Потренировался с принудительным выключением процессов Телеграма
Для начала открыл список всех процессов при помощи команды "ps -auxf"
Далее нашел процесс телеграмма, отметил его PID (например его значение было - 6845)
После этого при помощи команды "kill -9 6845" принудительно завершил процесс
Успешно!
